## Led-Matrix-UI

WebUI to manage HUB75 LED Matrix Panels with the hzeller/rpi-rgb-led-matrix library  

Build with:  
  * Go-Gin
  * Bootstrap
  * htmx

## Usage

Run as root to get access to the /dev GPIO
Configure it through the app flags
